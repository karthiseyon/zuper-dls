# Zuper — Design Language System

Self-contained HTML styleguide for Zuper. Open `index.html` (or the deployed URL).

Covers: **How to use this (Claude Design) · Deck · Website DLS · Case study** — colors, type
(Plus Jakarta Sans · Inter · Geist Mono · Aptos), single-weight line icons, components and slide layouts.

Deployed via Vercel.

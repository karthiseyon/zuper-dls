The official Zuper logomark (4-parallelogram "Z"), inline SVG so it recolors and scales cleanly.

```jsx
<Logo tone="color" size={40} />        {/* orange + charcoal, on light */}
<Logo tone="white" size={40} />        {/* on orange / photo grounds */}
<Logo tone="inverted" size={40} />     {/* orange mark on dark (Sense) */}
```

For the full horizontal/vertical lockup (mark + ZUPER wordmark), use the real files in `assets/logos/` — e.g. `zuper-horizontal.png` (light grounds) and `zuper-horizontal-white.png` (dark/orange grounds).
